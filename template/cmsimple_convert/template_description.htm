BETA :CMSimpleTEMPLATE TRANSLATOR

.. just for testing .. converts CMSimple Templates to cms.libe.net
cms.libe.net has nothing in common with CMSimple, this script only translates cmsimple templates to cms.libe.net

Usage: just copy the Template Files to ./templates/cmsimple_convert

how does it work: index.php calls and translates the copied file template.htm and its Images and CSS Style files

BETA: maybe there are some tags not correct interpreted, depending on the template
