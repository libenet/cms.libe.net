A simple website template from cms.libe.net
integrated Lightbox2 (jquery based)

includes the File phone-index.php, which loads an
alternative Layout optimized for Phones (including Smartphones)

if you like to build an Layout for Tablets create an tablet-index.php
(not only default, is possible for all templates)

to use the Phonelayout in another Template, simple copy the Files
phone-index.php and phone-main.css
to the folder of the other template
